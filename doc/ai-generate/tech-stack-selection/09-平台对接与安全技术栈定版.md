# 9. 平台对接与安全技术栈定版
## 9.1 接口规范落实
统一对外接口约定：

- `/api/v1` 版本前缀；
- JSON UTF-8；
- `code / msg / data / requestId` 响应结构；
- `enc` 签名字段；
- 可扩展的非必填字段机制。

## 9.2 鉴权与签名
建议技术实现：

- 平台来路请求：校验 `enc`、`time` 与必要身份字段；
- 内部会话：使用 `JWT` 或平台透传令牌；
- 权限控制：教师 / 学生 / 平台管理员使用 `RBAC`；
- 审计：关键接口写入调用日志与 `requestId`。

## 9.3 安全配套建议

- 所有外部调用默认 HTTPS；
- 敏感配置从环境变量读取；
- 课件资源、学习记录、问答记录按最小必要原则保存；
- 演示数据尽量使用脱敏样本。
